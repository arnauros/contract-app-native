Next.js Localhost Test Links

Instructions: Use these links to test your Next.js application's routing and subdomain handling. Click each link to see if it resolves correctly. The links open in a new tab.

If you see 404 errors, make sure:

  1. Your hosts file has entries for 127.0.0.1 localhost and 127.0.0.1 app.localhost
  2. Your Next.js server is running with npm run dev
  3. The middleware.ts file is correctly handling the subdomain routing
  4. The dashboard page exists at src/app/(dashboard)/dashboard/page.tsx
